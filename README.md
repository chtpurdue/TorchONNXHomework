# TorchONNXHomework
Brief explanation of whats included in this repository.
## Dataset
https://www.kaggle.com/datasets/samuelcortinhas/cats-and-dogs-image-classification/data
## index.html
The html for the assignment that is used for the webpage.
## main.js
The logic for the webpage itself. This is where the model is loaded and predictions are made using uploaded images.
## modeltorchonnx.ipynb
The code that was used to train and export the model.
## preprocessimagestorchonnx.ipynb
The code used for preprocessing the images.
## cnn_model_merged.onnx
The model that is used by the webpage. Images are passed into the model and predictions are made and spat out.
